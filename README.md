# ProgressPilot - Learning Tracker System
## Author: Abu Yahiya
## Author Email: rafiz.mehedi@gmail.com - +8801600221211

## Technology Used:-
* Nest.JS
* TypeScript
* MongoDB
* Mongoose
* GraphQL
* JWT - JsonWebToken

### Core Features:

1.  Learning Tracking
2.  Leader-Board based on progress
3.  Learning Report generation by day/week/month
4.  Reminder to learner for submit report
5.  Admin portal and Learner Portal

### Run commands
* npm/pnpm/yarn install
* npm run dev / pnpm/yarn dev# progress-pilot-api
